# ⚡ AI PPT 自动生成器

本项目基于 **FastAPI** 和 **SiliconFlow LLM**，提供一个完整的 AI PPT 自动生成服务：用户输入关键词 → LLM 生成标题和大纲 → 用户可修改 → 自动生成 PPT 并提供下载。

---

## 功能

1. **关键词生成标题和大纲**  
   - 输入关键词，AI 自动生成简洁、吸引人的 PPT 标题。
   - 自动生成 PPT 大纲，包含 3~5 个部分，每部分 2~4 个要点。

2. **用户可修改标题和大纲**  
   - 前端提供输入框和文本框，用户可自由修改 LLM 生成的内容。

3. **自动生成 PPT 文件**  
   - 根据用户修改后的标题和大纲生成 PPT。
   - 支持封面 + 多页内容，标题为每页标题，子弹点为每页内容。

4. **文件下载**  
   - 生成的 PPT 文件保存在 `generated/` 文件夹。
   - 提供下载链接或直接下载。
